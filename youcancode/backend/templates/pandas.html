1. Introduction

Pandas is a powerful Python library for data manipulation and analysis. It provides two main structures:

  • Series: One-dimensional labeled array
  • DataFrame: Two-dimensional labeled data table

2. Creating Structures

import pandas as pd

# Series
s = pd.Series([10, 20, 30])

# DataFrame
data = {'Name': ['Alice', 'Bob'], 'Age': [25, 30]}
df = pd.DataFrame(data)

2. Creating Data Structures

import pandas as pd

# Series
s = pd.Series([10, 20, 30])

# DataFrame from dictionary
data = {'Name': ['Alice', 'Bob'], 'Age': [25, 30]}
df = pd.DataFrame(data)
print(df)

3. Accessing Data

# Access column
df['Name']

# Access row by index
df.loc[0]       # by label
df.iloc[0]      # by integer position

4. Filtering, Sorting, Modifying

# Filtering
df[df['Age'] > 25]

# Sorting
df.sort_values(by='Age')

# Adding new column
df['Salary'] = [50000, 60000]

# Deleting column
df.drop('Salary', axis=1)

5. Handling Missing Values

df.isnull()          # Check missing values
df.dropna()          # Drop rows with missing data
df.fillna(0)         # Fill missing with 0

6. GroupBy and Aggregation

# Group by column
df.groupby('Name').mean()

# Aggregation
df['Age'].sum()
df['Age'].mean()

7. Merging and Concatenation

# Merge two DataFrames on common column
pd.merge(df1, df2, on='ID')

# Concatenate two DataFrames
pd.concat([df1, df2])

8. Reading / Writing CSV

# Reading from CSV
df = pd.read_csv('file.csv')

# Writing to CSV
df.to_csv('output.csv', index=False)
